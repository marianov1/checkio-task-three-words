بیاید به ربات ها یاد بدهیم که حروف را از اعداد تشخیص بدهند.

به شما به رشته (string) شامل کلمات و حروف داده شده که به وسیله فاصله -فضای خالی- از یکدیگر جدا شده اند. شما باید چک کنید که یکی از بخش های رشته (string) شامل سه کلمه باشد. برای مثال: "start 5 one two three 7 end" شامل یه کلمه در یک بخش است.

Input (ورودی): یک رشته (string) شامل چند کلمه

Output (خروجی): جواب در نوع بولین (boolean)

مثال: 

{% if interpreter.slug == "js-node" %}

threeWords("He is 123 man") == False
threeWords("1 2 3 4") == False
threeWords("bla bla bla bla") == True
threeWords("Hi") == False
{% else %}
checkio("Hello World hello") == True
checkio("He is 123 man") == False
checkio("1 2 3 4") == False
checkio("bla bla bla bla") == True
checkio("Hi") == False
{% endif %}

استفاده: به شما یاد میدهد که چگونه با رشته ها (string) کار کنید و همچنین شما را با چند تایع (function) پر کاربرد آشنا میکند.
