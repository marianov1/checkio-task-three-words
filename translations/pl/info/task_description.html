Nauczmy Roboty rozróżniać wyrazy i liczby.

Na wejściu jest napisz z wyrazami i liczbami oddzielonymi spacjami (po jednej). Wyrazy zawierają tylko litery. Sprawdź czy napis zawiera trzy wyrazy obok siebie (nie oddzielone liczbami). Na przykład napis "start 5jeden dwa trzy 7 koniec" zawiera trzy wyrazy obok siebie."

Wejście: Napis z wyrazmi i liczbami.

Wyjście: Odpowiedź jako wartość logiczna (boolean).

Przykłady: 

{% if interpreter.slug == "js-node" %}
threeWords("Hello World hello") == True
threeWords("He is 123 man") == False
threeWords("1 2 3 4") == False
threeWords("bla bla bla bla") == True
threeWords("Hi") == False
{% else %}
checkio("Hello World hello") == True
checkio("He is 123 man") == False
checkio("1 2 3 4") == False
checkio("bla bla bla bla") == True
checkio("Hi") == False
{% endif %}

Korzyści: To zadanie uczy Cię jak pracować z napisami i wprowadza użyteczne funkcje.

Założenia: Wejście zawiery wyrazy i/lub liczby. Nie ma mieszanych wyrazów (litery i liczby w jednym).
0 < len(wyrazy) < 100
